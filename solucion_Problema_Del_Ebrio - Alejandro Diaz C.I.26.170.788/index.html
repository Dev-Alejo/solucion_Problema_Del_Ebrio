Desarrollar una aplicación que permita resolver el problema del EBRIO


Realizado por: Alejandro Diaz C.I:26.170.788

Lenguajes utilizados: HTML y JavaScript


Supongamos que un ebrio está parado en una esquina, cuando decide caminar para que le pase el efecto “ya ustedes saben cuál”. Supongamos que existe una probabilidad igual de que se dirija al norte, sur, este u oeste al llegar a cada esquina. Si camina 10 calles, ¿Cuál es la probabilidad de que termine su recorrido a dos calles de donde lo empezó?


Para la solucion de este problema se utilizo el metodo Ramdom para la eleccion de un numero aleatorio con el cual poder evaluar la probabilidad




Numero cuadras recorridas Número Aleatorio Posicion X/Y

Si al finalizar el recorrido de las 10 cuadras la suma de X y Y resulta igual a 2 entonces el borracho terminó a una distancia igual a 2 dos calles de donde lo empezó.
